Jawaban Nomor 1:

Cara Kerja Website dan Hubungan antara Web Server, Domain, dan Hosting Cara Kerja Website: Pengguna (User): Mengakses website melalui browser. Browser: Mengirim permintaan HTTP ke server. Web Server: Menanggapi permintaan dengan mengirim halaman web (HTML, CSS, JavaScript, dll.). Halaman Web: Berkumpul menjadi tampilan dan fungsionalitas website. Hubungan antara Web Server, Domain, dan Hosting: Domain: Nama unik website di internet, didaftarkan melalui registrar. Hosting: Tempat penyimpanan file website, disediakan oleh penyedia hosting. Web Server: Program perangkat lunak di hosting yang merespons permintaan HTTP dan mengirim halaman web.
